Currency Intelligence Dashboard


Project Description -

This web application analyzes currency exchange rates using a public API. It allows users to view and interact with currency data to understand trends and relationships between different currencies.

API Used - 

Frankfurter API
https://api.frankfurter.app/latest

Features - 

Search currencies
Sort currencies based on exchange rates
Filter currency data
Trend analysis using historical data
Arbitrage detection

Technologies Used - 

HTML
CSS
JavaScript

Setup Instructions -

1. Clone the repository
2. Open the project folder
3. Open index.html in a browser
