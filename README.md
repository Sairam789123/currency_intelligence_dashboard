Currency Intelligence Dashboard

Project Description

This web application analyzes currency exchange rates using a public API. It allows users to view and interact with currency data to understand relationships between different currencies.

Purpose

The purpose of this project is to help users understand how different currencies behave instead of just converting values. It enables users to explore exchange rates and compare currencies in a simple and interactive way.

API Used

Exchange Rate API
https://open.er-api.com/v6/latest/USD

Features

Live exchange rate data
Search currencies (using `filter()`)
Sort currencies (High → Low, Low → High)
Dynamic base currency selection
View more functionality (loads additional currencies)
Clean and simple UI

Technologies Used

HTML
CSS
JavaScript

Setup Instructions

1. Clone the repository
2. Open the project folder
3. Open `index.html` in a browser
